■ 内容 (2.5Dアクション)
	地形を自動生成にすることで繰り返し遊べる、ローグライクとメトロイドヴァニアの融合を目指した作品です。
	
	フィールド内にはプレイヤーの身体能力を強化するアイテムが配置されており、地形を搔い潜りながら
	パワーアップアイテムを取得していくことがはじまりの目標となります。
	
	フィールドは円筒状になっており、左右どちらへも自由に進むことが出来ます。
	一見すると、円形ならば一方向に移動し続ければいずれループしてしまいそうですが、この作品では違います。
	移動するたびに常に新しい地形が登場し、通り過ぎた地形も変化していくので、直進し続けても必ず異なる様態を見せてくれます。
	たとえば、進みにくい地形が登場した場合は、一度引き返すことで有利な地形に変化する可能性も…？
	
	ただし、むやみやたらに移動してしまってはいけません。
	地形の変化に応じて、ゲージが蓄積していきます。このゲージが最大まで溜まり切ってしまうと…？
	変化し続けるフィールドの中でどれだけ鮮やかにアイテムを回収していけるのかが攻略のカギとなっています。

■ 操作方法

【キーボード操作】
	==================================================
	[タイトル]
		ゲーム開始 -> [ ENTER ]
	==================================================
	[インゲーム]
		移動 -----> [ A, D ]
		ジャンプ -> [ W, SPACE ]
		ダッシュ -> [ (移動時に)右クリック ]
		攻撃 -----> [ 左クリック ]
	==================================================
	[リザルト]
		タイトルに戻る -> [ ENTER ]
	==================================================

【コントローラー(XBOX)操作】
	==================================================
	[タイトル]
		ゲーム開始 -> [ A, メニューボタン ]
	==================================================
	[インゲーム]
		移動 -----> [ 左スティック, 十字キー左右 ]
		ジャンプ -> [ A ]
		ダッシュ -> [ (移動時に)X ]
		攻撃 -----> [ RT, Y ]
	==================================================
	[リザルト]
		タイトルに戻る -> [ A, メニューボタン ]
	==================================================
