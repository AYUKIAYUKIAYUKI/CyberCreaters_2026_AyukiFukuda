■ 内容 (2Dアクション)
	あなたは不思議な能力を持っている「メタモル」です。
	能力を駆使し、「メタモル」を操作して、ステージ内のゴールを目指してください。

	通常状態の「メタモル」はひたすら地面を歩くことしかできませんが、能力を発動することで空間を自由自在に飛び回れるようになります。
	地面の上にいる間は好きなタイミングで発動できますが、ひとたび飛び始めると何かにぶつかるまで止まることは出来ません。
	一長一短なこの能力を、使いこなせるかはあなた次第です。

	本作はステージ選択システムを採用しており、自由に選択できる複数のステージが用意されています。
	邪魔なブロックを壊してゴールを見つけ出すステージや、動き回る危険なトゲトゲを避けながらゴールにたどりつくステージ…
	全てのステージは共通してゴールを目指すことをクリア条件としています。
	また、各ステージごとにやりこみ要素となるアチーブメント回収、クリアタイムの記録が存在します。
	
	道筋を考えながら攻略するノーマルモードに加え、制限時間内にどれだけブロックを破壊できたかを競うチャレンジモードが存在します。
	チャレンジモードではプレイヤーが強化されており、一度飛び始めると衝突しても止まりません。
	サイズも巨大化し、突進中に方向転換も可能です。制約が解除された状態で、とにかく破壊しつくしましょう。

■ 操作方法

【キーボード操作】
	==================================================
	[タイトル]
		メニューへ -> [ ENTER ]
	==================================================
	[メニュー]
		モード選択 -> [ W, S ]
		モード決定 -> [ ENTER ]
	==================================================
	[ステージ選択]
		ステージ選択 -> [ A, D ]
		ゲーム開始 ---> [ ENTER ]
	==================================================
	[インゲーム]
		歩く -------> [ A, D ]
		とぶ -------> [ SPACE ]
		旋回 -------> [ (とびながら)W, A, S, D ]
		突進 -------> [ (とびながら)SPACE ]
		※リタイア -> [ BACK ]
	==================================================

【コントローラー(XBOX)操作】
	==================================================
	[タイトル]
		メニューへ -> [ A, B, X, Y, メニューボタン ]
	==================================================
	[メニュー]
		モード選択 -> [ 左スティック, 十字キー上下 ]
		モード決定 -> [ A, B, X, Y, メニューボタン  ]
	==================================================
	[ステージ選択]
		ステージ選択 -> [ 左スティック, 十字キー左右 ]
		ゲーム開始 ---> [ A, B, X, Y, メニューボタン  ]
	==================================================
	[インゲーム]
		歩く -------> [ 左スティック, 十字キー左右 ]
		とぶ -------> [ A, B, X, Y ]
		旋回 -------> [ (とびながら)左スティック, 十字キー ]
		突進 -------> [ (とびながら)A, B, X, Y ]
		※リタイア -> [ メニューボタン, ビューボタン ]
	==================================================

	※リタイアはノーマルモードのみ
	※数字マスならステージ開始、矢印マスならタイトルへ戻りアチーブメントをリセット
